# RhinoSpec

An (attempt at) distributed RSpec across multiple nodes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rhino_spec'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rhino_spec
